# Angry-BOX

ارکستر سبک فقط از طریق SSH. بک‌اند اصلی **sing-box**، ثانویه **xray**. بدون ایجنت روی نودها.

## پریست‌های استیلث ۲۰۲۶ (ویژگی اصلی)

بهترین شیوه‌های جامعه در میانهٔ ۲۰۲۶ (pumbaX، Xray XHTTP #4113، Hysteria Gecko، NaiveProxy و ...) را گرفته‌ایم و سیاست «امنیت مهم‌تر از سازگاری» را اجباری کردیم.

پریست‌های آماده:
- russia_2026 / iran_2026 / china_2026 / maximum_stealth_2026
- pro_2026 (CPS level 3 + QUIC 1200B اجباری)
- xhttp_max_stealth_2026

AWG با ژنراتورهای کامل I1-I5 عرضه می‌شود.

**قدردانی**: pumbaX، RPRX/Xray، Hysteria، NaiveProxy، Hiddify، 3x-ui و جامعهٔ تحقیقات پروکسی روسیه/ایران/چین.

## نصب (شامل پکیج‌های .ipk برای روتر)

```bash
# Keenetic (mipsel)
opkg install angry-box_0.2.0_mipsel_24kc.ipk

# OpenWRT aarch64
opkg install angry-box_0.2.0_aarch64_cortex-a53.ipk
```
