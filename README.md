# Angry-BOX

Лёгкий SSH-оркестратор для управления sing-box (по умолчанию) и xray (опционально) на удалённых машинах, включая Keenetic.

## Философия

- Оркестратор — это **только голова управления**. Он не является нодой цепочки.
- Управление — только через SSH. Без агентов на нодах.
- На удалённые машины (VPS, Keenetic и т.д.) ставится **только** прокси (sing-box или xray) + минимальный конфиг.
- Сам оркестратор можно поставить на Keenetic (для тех, у кого нет отдельного сервера) — в этом случае он выступает управляющей головой, но не участвует в цепочке как прокси (пока что).

## Два типа подключений

- **Транспортные** — технические линки между хопами внутри цепочки.
- **Пользовательские** — entry points для клиентов.
